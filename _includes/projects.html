HiceCoffee Project

HiceCoffe는 커피 판매 쇼핑몰 사이트입니다.
상품 구매를 비롯한 리뷰작성 및 다양한 인기원두와 오프라인 판매 매장의 위치까지 확인 가능하며, CRUD가 가능한 사이트 입니다.

참여인원 : 7명
개발기간 : 22.10.05~22.10.17

담당 기능
✔  조회수를 기준으로 한 인기상품 게시판 구현
✔  kakao 지도API 및 Spring Boot의 Thymeleaf문법과 Spring EL 표현식을 활용한 매장 안내 게시판, CRUD 기능 구현
✔  사용자들의 주문 정보 조회 가능한 관리자 페이지

Client  JAVA, JavaScript, CSS, HTML
Tool  Spring Framework
Server-File  Oracle DBMS

PETMILY

PEMILY는 유기견 입양 사이트입니다.
여러 지역의 유기견 정보를 확인 및 입양 신청서까지 비대면으로 작성하여 입양 신청을 할 수 있습니다.

참여인원 : 4명
개발기간 : 22.08.15~22.08.22

담당 기능
✔  입양 양식 내 사용자 입력 데이터 저장 기능 및 사용자 입력값 누락 시 alert로 안내 기능 구현
✔  유기견 입양 페이지 및 홈페이지 메인 UI 제작

Client  JavaScript, React, Redux, Axios, CSS, HTML
Tool  Visual Studio Code
Server-File  JSON
